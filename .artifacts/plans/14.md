## Schema mapping ‚ÄĒ common pattern per atom

```yaml
kind: font
id: <slug>
version: <semver>
name: <Human Name>
description: >
  <1‚Äď3 line summary of typeface + use case>
family: <Canonical CSS family>
classification: <serif|sans-serif|monospace>
tags: [ ‚Ä¶relevant slugs‚Ä¶ ]

provenance:
  source: <upstream repo URL>
  license: <SPDX>
  attribution: >
    <typeface>, Copyright <year> <Project Authors / designer>, licensed under <license>.
  importedDate: 2026-05-17
  importedFromVersion: <upstream release tag>
  notes: >
    <download URL or other context>

source:
  kind: <google-fonts|external>
  family: <CSS family>
  url:  <https URL if google-fonts>  # OR cssImportUrl for external
  notes: <only when external>

fallbackStack:
  - ‚Ä¶classification-appropriate stack‚Ä¶

availableStyles:
  - { weight: ‚Ä¶, style: normal }
  - ‚Ä¶
```

### Classification ‚Üí fallback stack policy

- `sans-serif`: `-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif`
- `serif`: `Georgia, Cambria, "Times New Roman", Times, serif`
- `monospace`: `Menlo, Monaco, Consolas, "Liberation Mono", "Courier New", monospace`

For Nerd-Font atoms, prepend the non-patched base mono first, then JetBrains Mono, then the
generic mono fallback ‚ÄĒ matching the pattern in `fonts/firacode-nerdfont/1.0.0/atom.yaml`.

### `source.kind` policy

- IBM Plex, Adobe Source, Mozilla Fira, Manrope, Plus Jakarta Sans, Public Sans, Work Sans,
  Crimson Pro: all available on Google Fonts ‚Üí `source.kind: google-fonts` with a real
  `fonts.googleapis.com/css2` URL. (`provenance.source` still points at the canonical
  upstream GitHub repo, not Google Fonts.)
- Nerd Font variants: `source.kind: external`, `notes` describing the Nerd Fonts release URL.

## Risk register

- **Designer attribution fidelity**: each atom carries the upstream OFL/Apache copyright
  line verbatim where known. Tester step grep-checks for fabrication markers.
- **Public Sans license**: upstream is currently OFL-1.1 (relicensed from CC0/USWDS-style

  release tag is `vN.MMM.P` (Adobe Source) we normalize to schema-compatible
  `MAJOR.MINOR.PATCH`.

  font; italic-bearing families get the `ital,wght@‚Ä¶` axis form.
- **Slug collision with #13**: covered above ‚ÄĒ different slug spaces (hyphenated vs
